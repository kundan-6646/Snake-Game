# Java-Snake-Game


<h3>Features</h3>
 <ul>
  <li>Created popular snake game using Java.</li>
  <li>Added different levels in game to make it interesting.</li>
  <li>Added pause/resume functionality.</li>
  <li>You can also view your score while playing the game.</li>
 </ul>
